## What is matchest?

matchest is a collection of scripts and tools for computational materials science, with three main components:

1. **CLI Tools**: Standalone command-line utilities for everyday tasks
2. **AiiDA Integration**: Automated workflows for high-throughput calculations
3. **Utilities**: Python modules for structure manipulation and analysis
